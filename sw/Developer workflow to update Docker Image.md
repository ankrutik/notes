Tags: #docker 

```
docker rm <docker_container_name>
docker rmi <docker_image_name>
docker build -t <docker_image_name>
docker run -d -p 5000:5000 --name <docker_container_name> <docker_image_name>
```

Steps:
1. Remove container
2. Remove image
3. Build image
4. Run image as container
